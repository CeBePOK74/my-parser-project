# VK Parser & Data Analyzer (Python/Scrapy)

[RU] 
Профессиональный инструмент для автоматизированного сбора и анализа данных из социальной сети ВКонтакте. 
Проект разработан с применением AI-технологий (DeepSeek/ChatGPT) для оптимизации кодовой базы и логики обхода защитных механизмов.

**Основные возможности:**
* Автоматический сбор постов и данных по заданным фильтрам.
* Интеграция с bat-файлами для быстрого запуска в Windows-среде.
* Логирование всех операций в реальном времени (`orders_log.txt`).
* Архитектура, оптимизированная под многопоточность на серверном оборудовании.

---

[EN]
A professional tool for automated data scraping and analysis from the VK social network. 
The project was developed using AI assistance (DeepSeek/ChatGPT) to optimize the codebase and anti-bot bypass logic.

**Key Features:**
* Automatic post and data collection based on custom filters.
* Bat-file integration for quick execution in Windows environments.
* Real-time operation logging (`orders_log.txt`).
* Architecture optimized for multi-threading on high-performance hardware.

## Stack:
* Python 3.x
* Scrapy / BeautifulSoup
* Windows Shell (Batch)
